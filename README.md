# 📘 Master Math Guide | Cálculo Diferencial

Una guía de estudio interactiva y minimalista diseñada para dominar los conceptos clave de Precálculo y Cálculo Diferencial. Incluye teoría directa, "hacks" matemáticos y un simulador de examen con clasificación de puntajes.

## ✨ Características Principales

*   **Diseño Moderno:** Interfaz limpia inspirada en las guías de diseño de Apple (Glassmorphism, animaciones suaves, tipografía nativa).
*   **Fórmulas Perfectas:** Integración con MathJax para renderizar código LaTeX y mostrar ecuaciones a nivel de libro de texto.
*   **Teoría Completa:** Leyes de potenciación, logaritmos, dominios, factorización y funciones.
*   **Hacks de Cálculo:** Trucos de agilidad mental para límites y derivadas (Regla de L'Hôpital, Regla de la Cadena).
*   **Simulador de Examen:** 10 preguntas aleatorias extraídas de un banco dinámico de 25 ejercicios.
*   **Leaderboard:** Sistema de clasificación Top 5 impulsado por `localStorage`.
